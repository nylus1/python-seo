Para poder ejecutar estos script en el terminal, necesitamos instalar Python.

Debemos ir a la Microsoft Store y buscar "Python 3.11". Lo instalamos y listo, ya podemos usarlo.

Cada Script puede usar librerias distintas, para instalarlas es tan fácil como escribir en el terminal "pip install + (nombre de la dependencia)". El nombre de estas dependencias esta escrito en el encabezado de cada script.

Se recomienda ejecutar los scripts desde VS Code.
